# Shilpa K. S. Portfolio

Personal portfolio built with Next.js App Router and a custom dark/light theme UI.

Hosted at: https://shilpaks.vercel.app/

## Tech Stack

- Next.js 16 (App Router)
- React 19
- TypeScript
- Tailwind CSS v4
- lucide-react icons

## Features

- App Router based tab navigation (Home, Experience, Projects, Contact)
- Responsive layout with sticky left profile panel on desktop
- Project filtering with Featured and Personal sections
- Dark and Light themes with manual toggle
- Optimized static build output

## Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
```
